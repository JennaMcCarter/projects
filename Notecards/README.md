# Creating a list of notecards
An Notecard project

User can...
* add a notecard
* edit notecard
* delete a notecard
* change color of notecard
* save notecard (and will auto save and show when it was last saved)
* export data
* import saved data
* start from the beginning
* move modal windows acround on the screen
